CLI -> Interfaz de Linea de comandos
Todos los comando de django se reliza mefiante el comando [django-admin]
django-admin startproject restaurante . > iniciar o crear un proyecto
python .\manage.py createsuperuser > crea un superusuario para que pueda utilizado en el panel administrativo
python .\manage.py runserver > levantar el proyecto en django
python .\manage.py showmigrations > para ver las migraciones
python .\manage.py migrate > para realizar las migraciones
python .\manage.py migrate nombre_migracion > para realizar las migraciones de la nombre_migracion.py
python .\manage.py startapp [nombre_app] > para crear una app No se puede crear aplicaciones con los nombres de las aplicaciones de Django: auth, contenttypes, message, users, admin, sessions, staticfiles
python .\manage.py makemigrations --help > para ver todas las opciones de la migracion
python .\manage.py makemigrations app --name creacion_tabla_etiquetas > para migrar todos los cambios de las tablas
python .\manage.py makemigrations NOMBRE_APP --name NOMBRE_MIGRACION > creara una nueva migracion de nuestra aplicacion con todos los cambios que se habran hecho en el ORM
python .\manage.py sqlmigrate NOMBRE_APP NUMERO_MIGRACION > mostrara el SQL de la migracion creada 
python .\manage.py migrate app 0001 > para revertir
python .\manage.py migrate app zero > para revertir todas las migraciones al principio
python .\manage.py dbshell > para ebrir una terminal de mysql
python .\manage.py flush > remueve toda la data de la DB
python .\manage.py test app.tests.EtiquetasTestCase.test_crear_etiqueta_success > ejecuta el test creado
python .\manage.py test app -v 2 > ejecuta el test creado detallado (0 - 3)
